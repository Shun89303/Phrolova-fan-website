
# Phrolova Fan Website

**Version:** 2.4 (Work-in-Progress)  
**Author:** Shun  
**Description:** A fan-made website dedicated to Phrolova from *Wuthering Waves*. This is a partially completed version with basic layout and partial styling.  

---

## Current Features

- **Background Music**  
  - Floating vertical button to play music.  
  - Music loops automatically.  

- **Falling Music Notes Animation**  
  - Music notes fall from the top of the screen when music is played.  
  - Animation is active but styling can be further refined.  

- **Navigation Bar**  
  - Sticky top navigation bar linking sections: About, Abilities, Gallery, Quotes.  
  - Partial styling applied.  

- **Header Section**  
  - Full-screen header with background image, title, subtitle, buttons, and quote.  
  - Partial styling applied.

---

## Work-in-Progress Sections

- **About, Abilities, Gallery, Voice Lines, Footer**  
  - Structure is complete in HTML.  
  - Minimal or no CSS styling applied yet.  
  - Will be fully styled in upcoming versions.  

---

## Installation / Usage

1. Clone the repository:  
```
  git clone https://github.com/Shun89303/Phrolova-fan-website.git
```
2.	Open index.html in a browser.
3.	Click the Play Music button to start the background audio and falling music notes animation.

⸻

File Structure

/ (root)
│─ index.html        # Main website structure
│─ styles.css        # Partial styling (nav + header mostly)
│─ script.js         # JavaScript: audio and falling music notes animation
│─ /images/          # All image assets
│─ /audio/           # Background music files
